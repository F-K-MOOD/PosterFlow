# 配置文件修改记录

## vite.config.ts

### 修改内容：
- **位置**：第14-23行（server配置部分）
- **修改前**：
```typescript
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:7001',
      changeOrigin: true,
      rewrite: (path) => path.replace(/^\/api/, ''),
    },
  },
}
```
- **修改后**：
```typescript
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:7001',
      changeOrigin: true,
      // 保留/api前缀，因为后端路由设置了前缀
      // rewrite: (path) => path.replace(/^\/api/, ''),
    },
  },
}
```

### 修改原因：
- 后端路由设置了`/api`前缀，但原代理配置会将`/api`前缀移除，导致请求路径不匹配
- 注释掉`rewrite`配置后，保持`/api`前缀，确保前端请求能正确匹配后端路由
